# vue2-admin

## Tech Stack

- Vue.js 2
- Vue Router 3
- Vuex 3
- Element UI 2
- ESLint + Prettier + lint-staged
- commitizen + commitlint
- pre-commit + commit-msg husky hooks

## Commands

### Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn dev
```

### Compiles and minifies for production

```bash
yarn build
```

### Commit Message

```bash
yarn cz
```
